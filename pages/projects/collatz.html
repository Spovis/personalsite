Collatz Conjecture visualizer

source code

What is it?

This is a visual representation of the Collatz conjecture. The essential foundation of the conjecture is that you start with a random positive integer, and then apply 2 rules.

If the number is even, you divide it by two.
If the number is odd, you multiply it by three and then add one.

You repeat this over and over. Go ahead, try it right now. For example, if you start with 9, you get the sequence 9, 28, 14, 7, 22, 11, 34, 17, 52, 26, 13, 40, 20, 10, 5, 16, 8, 4, 2, 1, 4, 2, 1, etc.
Theoretically, every possible starting number will eventually get stuck in a cycle of 4, 2, 1, but that has never been proven or disproven.

How does it work?

This visualizer picks random starting numbers and draws a graph of the path they take to reach that cycle. Each dendrite represents an individual starting number. After each step, the path it takes rotates slightly in a different direction based on whether it was an odd or even number.

For more info on the conjecture, see this video by Veritasium.
